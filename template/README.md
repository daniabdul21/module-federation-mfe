This is qcash app micro Frontend, build with Next.js + Module Federation.

## Getting Started
For setup this project, you can use yarn package manager for default package manager. First thing you have install all package using :

```bash
yarn install
```
And then run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# sao-qcash-ui-base-template

> template fe module federation

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao qcash-ui-base-template my-project
```

### From git

```bash
sao daniabdul21/sao-qcash-ui-base-template my-project
```
